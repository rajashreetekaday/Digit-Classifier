# Digit-Classifier
This project aimed to classify handwritten digits (0-9) using Neural Network and Convolutional Neural Network (CNN), using [MNIST Dataset](http://yann.lecun.com/exdb/mnist/).

### Hyperparameters
| Learning Rate    | 0.001     | 
| --------         | --------  | 
|**No. of epochs** | **5**     | 
| **Batch size**   | **64**    |
